
# CTH Tábor – Organizační repozitář

Tento repozitář slouží jako centrální místo pro organizaci, plánování a sdílení dokumentace k CTH astronomického tábora 2026.

## Účel repozitáře

- Uchovávání organizačních poznámek, úkolů a nápadů pro přípravu tábora
- Sdílení dokumentace a informací mezi členy organizačního týmu

## Struktura

- `docs/` – poznámky, zápisy, podklady a další dokumentace
- `README.md` – tento souhrnný popis repozitáře
- `mkdocs.yml` – konfigurace pro generování webové dokumentace

## Další informace

Podrobnější poznámky a aktuální informace najdete v adresáři `docs/`.
